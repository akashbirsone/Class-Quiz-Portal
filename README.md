# Class Quiz Portal

A modern and responsive quiz platform built with React and TypeScript, featuring real-time quiz management, student tracking, and AI-powered functionalities using Gemini API.

---

## 🚀 Live Demo
🔗 https://class-quiz-portal-ruby.vercel.app/ 

---

## ✨ Features

- 👨‍🎓 Student Profile Management
- 📝 Quiz System (MCQs with Timer)
- 📊 Result & Performance Tracking
- 🔐 Authentication (Login / Signup)
- ⚡ Fast UI with React + TypeScript
- 🎨 Modern UI using Tailwind CSS
- ☁️ Supabase Integration (Database + Auth)
- 🤖 AI Features (Gemini API)
  - Quiz generation / assistance
  - Smart suggestions (optional)

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- TypeScript
- Tailwind CSS

**Backend / Services:**
- Supabase (Database + Authentication)
